# NetFree Manual Link Checker

תוסף לבדיקה ידנית של רשימות קישורי YouTube מול הסינון של נטפרי.

## תכונות
- בדיקת רשימת קישורים גדולה בבת אחת.
- הדבקה מהלוח או טעינה מקובץ.
- אופציה לפתיחה בחלון נפרד.
- ייצוא תוצאות לקובץ או העתקה.
- ללא הרשאות מיותרות - בודק טקסט בלבד.
